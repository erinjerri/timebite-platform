# Roadmap

TimeBite's roadmap now prioritizes a calm habit loop over analytics-first dashboards.

## MVP Priority Order

1. Goal Loop
2. Daily Intent
3. Timer
4. Reflection
5. Monthly Completion Calendar
6. Dashboard
7. Analytics
8. AI features

---

## Phase 1: Goal Loop Foundation

- Define Goal domain model and persistence.
- Create, edit, pause, archive, and tag goals.
- Store frequency, completion history, sessions, reflections, and streak fields.
- Migrate old quarterly goals into Goal Loop records where possible.

## Phase 2: Daily Intent

- Add Today entry point.
- Let users choose one or more goals for the day.
- Keep intent entry plain-language and low-friction.
- Show completion checkmarks directly in Today.

## Phase 3: Timer Execution

- Connect timer sessions to goals.
- Support start, pause, resume, complete, partial, skip, and rest day.
- Persist session recovery locally.
- Update completion history from completion events.

## Phase 4: Reflection

- Add post-session reflection prompt.
- Support one-line reflection and optional mood/tag.
- Tie reflections to goal and session.
- Allow reflection later from calendar day detail.

## Phase 5: Monthly Completion Calendar

- Render month grid by goal and aggregate view.
- Show completion, partial, skipped, rest, today, and future states.
- Open day detail with sessions and reflections.

## Phase 6: Dashboard

- Summarize current streak, longest streak, weekly summary, monthly summary, and recent reflections.
- Keep dashboard calm and scannable.
- Avoid dense charts until analytics phase.

## Phase 7: Analytics

- Reintroduce deeper metrics as derived insight projections.
- Map session data into time/category views only after core retention is working.
- Keep analytics explainable from completion and reflection data.

## Phase 8: AI Features

- Add constrained suggestions only after explicit privacy review.
- Support reflection synthesis, gentle prompts, and goal suggestions.
- Do not require AI for the core loop.
